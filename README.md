## üìå Overview

**Schulde 1 Mod Menu** is a Python-based console save editor that allows users to view and modify game save data such as bank money, inventory items, property storage, and pot data.

The tool works by directly editing local save files and is intended for offline or personal gameplay use.

## üõ†Ô∏è How to Use

### Step 1: Locate Save Files
1. Download **`schulde1modmenu.exe`**  
2. Press **Win + R**  
3. Type:**`appdata`**
4. Navigate to: locallow-> TVGS ‚Üí Schulde 1 ‚Üí Saves
5. Open the folder with **random numbers**  
6. Select your **game save folder**  
7. Use this path in the mod menu when prompted

---

### Step 2: Update Game Data (IMPORTANT)
Before making any changes:

1. Save the game using the **buttons at your property**  
**OR**  
2. Go to **sleep** in-game  

This ensures the save files contain **updated data**.

---

### Step 3: Modify Save Data
1. Open the mod menu  
2. **View** the data you want to change to confirm it is updated  
3. **Modify** the desired values  
4. **Quit the game**

---

### Step 4: Apply Changes
1. Launch the game again  
2. Your changes should now be applied
